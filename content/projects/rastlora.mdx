# Rast LoRa

Rastreador veicular com utilização de rede lora para comunicação. Com rede em comunidade para aumento de alcance e coleta de informações sobre a saúde do veículo.

## Introdução

O projeto Rast LoRa tem como objetivo a criação de um rastreador veicular com utilização de rede lora para comunicação. Com rede em comunidade para aumento de alcance e coleta de informações sobre a saúde do veículo.

O projeto utliza de comunicação lora como rede principal mas para casos em que a rede lora não esteja disponível, o projeto conta com um módulo GSM para comunicação.

## Materiais

- ESP32 LoRa 868/915 Mhz com display OLED
- Módulo GSM SIM800L
- Sensor de temperatura e umidade DHT11
- Módulo GPS NEO-6M
