# Desapega Conquista

O desapega conquista é um software idealizado para solucionar uma falta de tecnologia para pequenos e medianos comerciantes e consumidores de uma cidade. A ideia é criar um aplicativo de e-commerce B2C e C2C, que possa ser utilizado por qualquer pessoa, sem a necessidade de conhecimento técnico.

O aplicativo possui um sistema de whitelabel para comerciantes que permite aos mesmos terem sua propia loja online com apenas os seus produtos em sua propia tematica e ao mesmo tempo ter seus produtos disponiveis no marketplace do aplicativo.

## Tecnologias

O aplicativo foi desenvolvido utilizando as seguintes tecnologias:

- React Native (Expo)
- Node.js

Backend:

- Express
- MongoDB
- Node.js
- TypeScript
- Aws S3
- API Whatsapp
- API de pagamentos (celcoin, pagar.me)

Frontend:
- Next.js
- React
- TypeScript
- MUI
- Axios
- PWA - Progressive Web App

Aplicativo publicado na [Play Store](https://play.google.com/store/apps/details?id=com.digitalmoontech.desapegaconquista)
